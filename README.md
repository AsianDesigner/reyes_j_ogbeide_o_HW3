# Taste of Africa

A simple and elegant web application that highlights the diverse and rich culinary traditions of Africa. Built with Vue.js, JavaScript, and a Lumen (PHP) backend API.

## Features

- Interactive UI built with Vue.js
- Dynamic data fetching via Axios
- Backend powered by Lumen
- Highlights various African dishes with images and descriptions
- Fully responsive design

## Technologies

- Frontend: Vue.js, JavaScript, HTML, Sass/CSS
- Backend: Lumen (PHP)
- API Requests: Axios
